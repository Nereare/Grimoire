# 3.5e Item Example

Here is an example of an empty item entry.

```yaml
---
title        : Item Page Title
author       : Author Name
date         : 2020-09-11

item:
  name: Item Name
  type: Ring
  weight: 2 lb.
  value: 50.5
  attributes:
    - Advantage on Wisdom saving throws
    - +2 AC

layout: item
---

# Description of the item here.
```
