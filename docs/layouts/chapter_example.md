# Chapter Example

Here is an example of an empty chapter entry.

```yaml
---
title      : Chapter Title
author     : Author Name
date       : 2019-08-19

chapter:
  book     : Some Book Name
  number   : 1
  previous : false
  next     : next_chapter

layout     : chapter
---

# Chapter content here.
```
