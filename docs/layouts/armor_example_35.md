# 3.5e Armor Example

Here is an example of an empty armor item entry.

```yaml
---
title        : Armor Page Title
author       : Author Name
date         : 2020-09-12

item:
  name: Armor Name
  type: Heavy
  weight: 84 lb.
  value: 8,000
  ac:
    base: 18
    dex: 0
  e35:
    str: 16
    penalty: 0
  attributes:
    - Can cast thaumaturgy

layout: armor
---

# Description of the armor here.
```
